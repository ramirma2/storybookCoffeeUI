## Accessibility

- Always provide a visible or accessible label (`aria-label` or `<label>`).
- Ensure error messages are programmatically associated with the input via `aria-describedby`.
- Use semantic `type` attributes (e.g., `email`, `password`, `number`) for better browser behavior.
